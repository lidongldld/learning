## 1.人民监督员人员信息

#### 接口信息

接口名：/api/procuratorate/supervisorInfo

##### 请求方式：

请求头校验:

```json
{
    "AppId":"账号的appId",
    "Nonce":"当前时间戳字符串"，
    "Sign":"签名(nonce+appSecret拼接字符串，然后使用HmacSha256方法签名并进行Base64编码)，如Base64.encodeBase64String(HmacUtils.hmacSha256(APP_SECRET, nonce + APP_SECRET))"
}
```



##### 请求参数



| 字段名 | 是否必须 | 描述                       |
| :----: | :------: | -------------------------- |
|   id   |    否    | id传返回个人，不传返回全部 |



```json
{
    "id":""
}
```
